# lambda-zsh-theme
Lambda ZSH Theme

This ZSH theme is a customized combination of two themes:
* [Oh-My-Fish lambda theme] 
* [lambda-mod-zsh-theme]

The main features of this theme are:
* Git-aware
* Uses 256 colors
* Multiline prompt with lambda symbol
